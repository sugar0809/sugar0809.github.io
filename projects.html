PROJECTS

ユーザの進捗に合わせた調理ナビゲーションシステム：MimiCook（2013~2014）

MimiCookは、ユーザの調理の進捗に応じて、キッチンカウンター上にナビゲーションをするシステムです。
MimiCook embodies a recipe in a real kitchen counter and directly navigates a user.

リアルタイムフィードバックをするスマートまな板：CookTab（2012）

CookTabは、食材の切り方や重量を記録したり、食材を切るたびにリアルタイムフィードバックが表示されるスマートまな板です。
CookTab allows a user to easily record his/her cooking activities while provides real-time feedback.
詳細 / detail

計らなくて済むスプーン : smoon（2011）

smoonは、必要な分量に合わせて入る容量が変化する計量スプーンです。
smoon is a smart measuring spoon that changes its volume according to required amount.
動画 / Movie

眺めるファッションコーディネート支援システム : suGATARIUM（2011）

suGATARIUM(スガタリウム)は、suGATALOGで記録した毎日の服装の写真をファッションショー形式で閲覧するアプリケーションです。
suGATARIUM is an app that helps you to decide what to wear by watching fashion show animation with using fashion lifelogs.

大縄跳びとオーケストラを融合したエンタメシステム：大縄オーケストラ（2010）

大縄オーケストラは、バーチャルな大縄跳びを跳ぶと、参加人数に応じた演奏ができるエンターテイメントシステムです。
"The Group Jump-Rope Orchestra" is an entertainment exercise system jumping a virtual rope.

二人三脚でバーチャルに旅ができるスポーツシステム：何人何脚!?（2010）

何人何脚!?は、二人三脚のように足踏みを踏むとバーチャルに旅ができる協調型の運動システムです。
"How many legged races!?" is a virtual exercise system by stepping.

姿を利用したファッションコーディネート支援システム : suGATALOG（2008-2010）

suGATALOG(スガタログ)は、毎日の服装を記録し、画面上でコーディネートを行うことができるアプリです。
suGATALOG is a fashion coordination app.

時間割型ファイル管理システム : Drop Table（2009）

Drop Tableは授業の時間割に特化したファイル管理アプリです。
Drop Table is a filing system focused on school timetables.
